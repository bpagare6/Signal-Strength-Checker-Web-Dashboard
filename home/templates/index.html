Easy And Perfect Solution For Analysing Signal Strength all across the Country

We built this app with a vision of analysing signal strengths across our country and allowing different network providers to take suitable actions to improve the connectivity. This will also allow our government to keep track of coverage by different operators very easily and efficiently.

To solve this problem we built an Android App and a web application. Android app collects the signal strength along with location information and stores it in the database which is then used by web application to show the connectivity dashboard.
